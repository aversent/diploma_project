# Tree Editor 

Веб-приложение для построения графов и деревьев с использованием Python, Dash и Cytoscape.  
Позволяет добавлять узлы, менять их цвет, загружать изображения и сохранять граф в JSON.  

 Технологии  
- Python  
- Dash  
- Dash Cytoscape  
- Bootstrap Components
